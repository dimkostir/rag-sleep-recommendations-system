## 📌 Project Overview  

This project is a **Sleep Quality Tracking Application** designed to help users **monitor, analyze, and improve** their sleep habits.

The system collects sleep data either:
- automatically (via wearable device APIs), or  
- manually (through user input)

It then evaluates the data using:
- a **sleep scoring mechanism**, and  
- an **AI-powered agent**

to generate **personalized insights and recommendations** for improving sleep quality.

---

## 🚀 Key Features  

- 🔐 **User Authentication** (Register / Login with JWT)  
- 🛌 **Sleep Data Tracking** (manual input & API integration)  
- 📊 **Sleep Quality Scoring System**  
- 🤖 **AI-Powered Recommendations** (LangChain)  
- 📚 **Scientific Knowledge Base Integration** (FAISS)  
- 🧠 **Personalized Feedback & Insights**  
- 🗄️ **MongoDB Data Storage**  
- 📅 **30-Day Sleep History per User**

---

## 🧠 Technologies Used  

- **Backend:** FastAPI (Python)  
- **Frontend:** React  
- **Database:** MongoDB  
- **AI Processing:** LangChain + Groq API  
- **Vector Storage:** FAISS  

---

## 🎯 Goal  

The goal of this project is to build an intelligent system that not only tracks sleep data but also **understands user behavior** and provides **personalized, science-based recommendations** to improve overall sleep health.
