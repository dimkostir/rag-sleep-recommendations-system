📌 Project Overview

This project is a Sleep Quality Tracking Application designed to help users monitor, analyze, and improve their sleep habits. It combines modern web technologies with intelligent data processing to deliver personalized insights and recommendations.

The system collects sleep-related data either automatically (via wearable device APIs) or manually through user input. It then evaluates this data using a scoring mechanism and an AI-powered agent to provide meaningful feedback and suggestions for better sleep quality.

🚀 Key Features
User authentication (Register / Login with JWT)
Sleep data tracking (manual input or API integration)
Sleep quality scoring system
AI-powered recommendations using LangChain
Integration with a scientific knowledge base
Personalized feedback and insights
Data storage with MongoDB
30-day sleep history tracking per user
🧠 Technologies Used
Backend: FastAPI (Python)
Frontend: React
Database: MongoDB
AI & Processing: LangChain, Groq API
Vector Storage: FAISS
🎯 Goal

The main goal of this project is to create an intelligent system that not only tracks sleep data but also understands user behavior and provides personalized, science-based recommendations to improve overall sleep health.
